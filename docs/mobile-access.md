# モバイルデバイスからのアクセス方法

## 概要
スマートフォンやタブレットなど、他のデバイスからアプリにアクセスするための設定方法です。

## 設定手順

### 1. ホストマシンのIPアドレスを確認
```bash
hostname -I | awk '{print $1}'
```

### 2. 環境変数を設定
`.env.local`ファイルを作成または編集：
```bash
NEXT_PUBLIC_API_URL=http://[ホストIP]:8080
```

例：
```bash
NEXT_PUBLIC_API_URL=http://192.168.151.145:8080
```

### 3. コンテナを再起動
```bash
docker-compose down
docker-compose up -d
```

## アクセス方法

### フロントエンド（Next.js）
```
http://[ホストIP]:3000
```

### API（Go）
```
http://[ホストIP]:8080
```

## ファイアウォール設定

Linuxの場合、必要に応じてポートを開放：
```bash
sudo ufw allow 3000/tcp  # フロントエンド
sudo ufw allow 8080/tcp  # API
```

## トラブルシューティング

### 1. 接続できない場合
- ホストマシンのファイアウォール設定を確認
- 同じネットワーク（WiFi）に接続していることを確認
- IPアドレスが正しいことを確認

### 2. 画像が表示されない場合
- API URLの設定が正しいことを確認
- ブラウザの開発者ツールでネットワークエラーを確認

### 3. 開発時のみの注意点
- HTTPSではないため、一部のブラウザで制限がある場合があります
- 本番環境ではHTTPS化が必要です
