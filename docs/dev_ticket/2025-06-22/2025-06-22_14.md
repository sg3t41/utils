## 概要

記事にアイキャッチ画像をアップロード・表示する機能を実装する。AWS無料枠内で運用するため、EC2のEBSボリューム（30GB）内の/uploadsディレクトリを使用する。

## 背景・目的

- 記事のビジュアル性を向上させる
- アイキャッチ画像による記事の魅力向上
- AWS無料枠内での実装を実現
- EC2直接デプロイでコスト最適化

## 技術詳細

### 対象システム・モジュール
- `/api/internal/interfaces/handler/` - 画像アップロードハンドラー
- `/api/internal/application/usecase/` - 画像処理UseCase
- `/frontend/app/articles/` - 記事管理画面
- `/uploads/images/` - 画像保存ディレクトリ

### 実装方針
1. **セキュリティファースト**: ファイル検証の徹底
2. **軽量化**: 画像の自動最適化
3. **ユーザビリティ**: 直感的なアップロード体験
4. **保守性**: 設定可能な制限値

### 技術スタック
- **バックエンド**: Go (Gin), 画像処理ライブラリ
- **フロントエンド**: Next.js, React, ドラッグ&ドロップライブラリ
- **ストレージ**: ローカルファイルシステム (/uploads)
- **画像処理**: Go標準ライブラリ + サードパーティ

### ディレクトリ構造
```
/uploads/
├── images/
│   ├── articles/
│   │   ├── 2025/
│   │   │   ├── 06/
│   │   │   │   ├── 22/
│   │   │   │   │   ├── uuid-original.jpg
│   │   │   │   │   └── uuid-thumbnail.jpg
```

### データベース変更
```sql
ALTER TABLE articles ADD COLUMN featured_image VARCHAR(500);
ALTER TABLE articles ADD COLUMN thumbnail_image VARCHAR(500);
```
