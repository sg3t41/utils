## 概要

記事新規作成ページ（`/articles/new`）で「useEffect is not defined」エラーが発生し、ページが正常に表示されない。

## 背景・目的

リファクタリング後の動作確認中に発見。記事作成機能が完全に使用不可能な状態。

## バグ詳細

### エラー内容
```
ReferenceError: useEffect is not defined
    at NewArticlePage (page.tsx:32:3)
    at ClientPageRoot (client-page.tsx:60:12)
```

### 根本原因
`frontend/app/articles/new/page.tsx`の3行目でReactフックのインポートが不完全：

**現在（問題）:**
```typescript
import { useState } from 'react';
```

**32行目で使用:**
```typescript
useEffect(() => {
  setMounted(true);
}, []);
```

### 影響範囲
- 記事新規作成ページが完全に表示されない
- 404エラーページが代わりに表示される
- 記事作成機能が使用不可能
